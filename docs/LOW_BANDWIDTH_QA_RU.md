# Проверка на слабом интернете

Где белые списки? должен быть полезен именно тогда, когда интернет плохой.

## Что проверить

- Главная страница открывается на мобильном.
- Пользователь видит последние статусы без долгого ожидания карты.
- Форма отчета доступна без регистрации.
- Текст не наезжает на кнопки.
- Карта не блокирует доступ к списку.
- Если не загрузились тайлы карты, список все равно работает.
- Нет внешних шрифтов.
- Нет тяжелых баннеров и виджетов.
- Страницы правил/FAQ открываются быстро.

## Устройства

- Android Chrome.
- iPhone Safari, если доступен.
- Десктопный браузер.
- Мобильная сеть.
- Домашний интернет.

## Условия

- Обычное соединение.
- Медленное соединение.
- Перезагрузка страницы.
- Повторное открытие.

## Критерии

- Первый полезный экран: до 3 секунд на нормальной мобильной сети.
- Отправка отчета: до 60 секунд у нового пользователя.
- Без карты пользователь все равно понимает текущую ситуацию.
