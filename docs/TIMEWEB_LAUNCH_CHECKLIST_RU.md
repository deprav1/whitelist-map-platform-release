# Чеклист запуска на Timeweb

## До SSH

- Выбран домен или поддомен.
- Понятно, какой тариф Timeweb используется.
- Известно, есть ли SSH.
- Известно, есть ли PHP CLI.
- Известна версия PHP.
- Проверено наличие MySQL.
- Проверено наличие cron.
- Проверена поддержка `.htaccess`.

## Файлы

- Собрать пакет:

```powershell
.\scripts\package-timeweb-shared.ps1 -SiteUrl 'https://your-domain.ru' -Force
```

- Загрузить содержимое `public_html/` из архива в корень сайта.
- Проверить, что `.htaccess` загрузился, а не потерялся из-за скрытого имени.

## База

- Создать MySQL-базу.
- Создать пользователя.
- Записать host, database, username, password.
- Создать `platform/.env` из `platform/.env.timeweb.example`.
- Вписать реквизиты БД.

## Миграции

```bash
cd /home/<user>/<site-root>/platform
php ./bin/phinx migrate -c phinx.php
chmod -R 775 storage
```

## Cron

Добавить задачи из `docs/TIMEWEB_DEPLOY.md`.

## Bootstrap

С локальной машины:

```powershell
$env:USH_BASE_URL='https://your-domain.ru'
.\scripts\bootstrap-whitelist.ps1
```

## Безопасность

- Сменить пароль администратора.
- Проверить, что публичная форма не требует регистрации.
- Проверить, что геоточка необязательная.
- Проверить, что отчеты уходят на модерацию.
- Опубликовать ссылки на приватность, правила и FAQ.

## Первый тест

- Отправить отчет.
- Проверить очередь модерации.
- Удалить персональные данные из тестового комментария, если есть.
- Опубликовать отчет.
- Проверить, что он виден публично.
